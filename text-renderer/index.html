This is a testing program for CS 31 project 5.
It generates many sample inputs and checks that your program's output matches the official tester.

Usage

  1. SSH into SEASnet. This code has been confirmed to work on lnxsrv11.seas.ucla.edu.
  2. cd into the directory where your render.cpp is. Confirm that you can build and run render.cpp under g31.
  3. Edit your render.cpp file: comment out the main function and write this at the bottom:
    4. #include <iostream>
int main() {
    int lineLength;
    std::cin >> lineLength;
    int returned = render(lineLength, std::cin, std::cout);
    std::cout << '\n' << returned;
}
  4. Run the following shell commands in the directory containing render.cpp: 
    $ 
    wget https://tombinford.github.io/text-renderer/testing-script.py
    $ 
    chmod +x testing-script.py
    $ 
    g31 -o render-tweaked render.cpp && ./testing-script.py ./render-tweaked
  5. Debug any failing inputs reported by the script. Good luck!
